# 원소들의 곱과 합

## 문제 설명
정수가 담긴 리스트 `num_list`가 주어질 때, 모든 원소들의 곱이 모든 원소들의 합의 제곱보다 작으면 1을 크면 0을 return하도록 solution 함수를 완성해주세요.

## 제한사항
- 2 ≤ `num_list`의 길이 ≤ 10
- 1 ≤ `num_list`의 원소 ≤ 9

## 입출력 예
| num_list | result |
| :-- | :-- |
| [3, 4, 5, 2, 1] | 1 |
| [5, 7, 8, 3] | 0 |

## 입출력 예 설명
입출력 예 #1
- 모든 원소의 곱은 120, 합의 제곱은 225이므로 1을 return합니다.

입출력 예 #2
- 모든 원소의 곱은 840, 합의 제곱은 529이므로 0을 return합니다.

## 이전 학습 내용 적용
- 이전에 학습했던 `reduce`, `pow` 메서드를 사용해서 문제를 해결하였다.

## 나의 풀이
```swift
func solution(_ num_list:[Int]) -> Int {
    num_list.reduce(1, *) < Int(pow(Double(num_list.reduce(0, +)), 2.0)) ? 1 : 0
}
```

## 배운점
- `reduce` 메서드의 초기값을 덧셈 연산 시에는 0을 초기값으로 사용하고, 곱셈 연산 시에는 초기값을 1로 설정을 해야 하는 연산의 항동원 이라는 것을 배웠다.
- `reduce` 메서드의 초기값을 튜플 형태로 설정하여 배열을 여러번 탐색하지 않고 한번의 탐색으로 여려개의 값을 동시에 추출하는 방법을 배웠다.
- `pow` 메서드를 사용하기 위해 타입 변환대신 정수의 곱셈을 활용하여 가독성과 런타임 오차 가능성을 방지했다.

## 새로운 풀이
```swift
func solution(_ num_list:[Int]) -> Int {
    
    let result = num_list.reduce((multi: 1, sum: 0)) { (state, next) in
        (state.multi * next, state.sum + next)
    }
    
    return result.multi < result.sum * result.sum ? 1 : 0
}
```
