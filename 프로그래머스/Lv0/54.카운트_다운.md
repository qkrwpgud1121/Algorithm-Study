# 카운트 다운

## 문제 설명
정수 `start_num`와 `end_num`가 주어질 때, `start_num`에서 `end_num`까지 1씩 감소하는 수들을 차례로 담은 리스트를 return하도록 solution 함수를 완성해주세요.

## 제한사항
- 0 ≤ `end_num` ≤ `start_num` ≤ 50

## 입출력 예
| start_num | end_num | result |
| :-- | :-- | :-- |
| 10 | 3 | [10, 9, 8, 7, 6, 5, 4, 3] |

## 입출력 예 설명
입출력 예 #1
- 10부터 3까지 1씩 감소하는 수를 담은 리스트는 [10, 9, 8, 7, 6, 5, 4, 3]입니다.

## 이전 학습 내용 적용
- 이전 문제에서 학습했던 `stride`를 사용하여 오름차순 뿐만 아니라 요번 문제에서 사용했던 `by: -1`을 사용하여 내림차순을 적용하였다.
- `(end_num...start_num).reversed()` 를 사용하여 범위를 먼저생성하고 뒤집는 방식도 가능하지만 오름차순 메모리를 사용했다가 뒤집어야하는 추가적인 연산 처리가 필요하다.
- 해서 `stride`을 사용하여 처음부터 내림차순으로 배열을 생성하여 불필요한 연산을 최대한 하지 않도록 하였다.

## 나의 풀이
```swift
func solution(_ start_num:Int, _ end_num:Int) -> [Int] {
    return Array(stride(from: start_num, through: end_num, by: -1))
}
```
