# n개 간격의 원소들

## 문제 설명
정수 리스트 `num_list`와 정수 `n`이 주어질 때, `num_list`의 첫 번째 원소부터 마지막 원소까지 `n`개 간격으로 저장되어있는 원소들을 차례로 담은 리스트를 return하도록 solution 함수를 완성해주세요.

## 제한사항
- 5 ≤ `num_list`의 길이 ≤ 20
- 1 ≤ `num_list`의 원소 ≤ 9
- 1 ≤ `n` ≤ 4

## 입출력 예
| num_list | n | result |
| :-- | :-- | :-- |
| [4, 2, 6, 1, 7, 6] | 2 | [4, 6, 7] |
| [4, 2, 6, 1, 7, 6] | 4 | [4, 7] |

## 입출력 예 설명
입출력 예 #1
- [4, 2, 6, 1, 7, 6]에서 2개 간격으로 저장되어 있는 원소들은 [4, 6, 7]입니다.

입출력 예 #2
- [4, 2, 6, 1, 7, 6]에서 4개 간격으로 저장되어 있는 원소들은 [4, 7]입니다.

## 배운점
- `stride`를 사용하여 필요한 인덱스의 시퀸스만 지연 생성한뒤 `map`을 체이닝 하여 문제를 해결하였다.
- `stride`사용시 상한선을 포함하는 `through`대신 포함하지 않는`to`를 사용하여 Index Out of Range 를 방지 하였다.
- `stride`사용시에 시작과 끝 값을 0과 .count로 하지 않고 `startIndex`, `endIndex`를 사용하여 인덱스가 0부터 시작하지 않을 수도 있는 상황에서 안전하게 동작하도록 하였다.

## 나의 풀이
```swift
func solution(_ num_list:[Int], _ n:Int) -> [Int] {
    stride(from: num_list.startIndex, to: num_list.endIndex, by: n).map { num_list[$0] }
}
```
