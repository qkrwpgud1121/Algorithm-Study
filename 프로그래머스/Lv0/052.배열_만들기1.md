# 배열 만들기 1

## 문제 설명
정수 `n`과 `k`가 주어졌을 때, 1 이상 `n`이하의 정수 중에서 `k`의 배수를 오름차순으로 저장한 배열을 return 하는 solution 함수를 완성해 주세요.

## 제한사항
- 1 ≤ `n` ≤ 1,000,000
- 1 ≤ `k` ≤ min(1,000, n)

입출력 예
| n | k | result |
| :-- | :-- | :-- |
| 10 | 3 | [3, 6, 9] |
| 15 | 5 | [5, 10, 15] |

## 입출력 예 설명
입출력 예 #1
- 1 이상 10 이하의 3의 배수는 3, 6, 9 이므로 [3, 6, 9]를 return 합니다.

입출력 예 #2
- 1 이상 15 이하의 5의 배수는 5, 10, 15 이므로 [5, 10, 15]를 return 합니다.

## 배운점
- `sorted`의 불필요한 연산 제거 - 1...n 의 범위는 오름차순으로 생성이 되기때문에 .filter를 사용한 이후에도 오름차순을 유지한다 결국 .sorted를 또 한번 붙이는것은 추가적인 연산을 낭비하는 불필요한 오버헤드 라는것을 알고 제거하였다.
- 전체 탐색과 `stride`탐색
    - 첫번째 풀이처럼 filter를 사용하면 1부터 n번째 까지 isMultiple에대한 연산을 해야한다.
    - 두번째 풀이처럼 `stride`를 사용하면 처음부터 k만큼 건너뛰며 값을 생성하기때문에 연산을 많이 줄일 수 있다.

## 나의 풀이

### 첫번째 풀이
```swift
func solution(_ n:Int, _ k:Int) -> [Int] {
    (1...n).filter { $0.isMultiple(of: k) }
}
```

### 두번째 풀이
```swift
func solution(_ n:Int, _ k:Int) -> [Int] {
    Array(stride(from: k, through: n, by: k))
}
```
