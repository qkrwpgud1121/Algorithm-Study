# 문자열 붙여서 출력하기

## 문제 설명
두 개의 문자열 `str1`, `str2`가 공백으로 구분되어 입력으로 주어집니다.
입출력 예와 같이 `str1`과 `str2`을 이어서 출력하는 코드를 작성해 보세요.

## 제한사항
- 1 ≤ `str1`, `str2`의 길이 ≤ 10

## 입출력 예

**입력 #1**
```
apple pen
```
**출력 #1**
```
applepen
```
**입력 #2**
```
Hello World!
```
**출력 #2**
```
HelloWorld!
```
---

## 나의 풀이
```swift
import Foundation

let inp = readLine()!.split(separator: " ")
print(inp[0]+inp[1])
```

## 이전 학습 내용 적용
- 문제를 보고 전에 사용했던 split(separator:) 를 사용하면 공백이 제거 된다는것을 알고있었기 때문에 문제 해결에 큰 도움이 되었다.

## 배운점
- inp[0]+inp[1] 이렇게 하면 입력받은 문자열의 첫번째와 두번째에 국한되어 만약 많은 문자열을 입력받았다면 전체의 문자열을 합쳐서 보여줄수 없다. 해서 만약 여러개의 문자열을 입력받았을때 입력받은 문자열을 전부 합하기 위해서는 joined() 를 사용하면 된다.

## 새로운 풀이
```swift
let inp = readLine()!.split(separator: " ")
print(inp.joined())
```
