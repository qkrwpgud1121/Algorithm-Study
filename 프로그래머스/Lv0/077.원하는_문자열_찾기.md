# 원하는 문자열 찾기

## 문제 설명
알파벳으로 이루어진 문자열 `myString`과 `pat`이 주어집니다. `myString`의 연속된 부분 문자열 중 `pat`이 존재하면 1을 그렇지 않으면 0을 return 하는 solution 함수를 완성해 주세요.

단, 알파벳 대문자와 소문자는 구분하지 않습니다.

## 제한사항
- 1 ≤ `myString`의 길이 ≤ 100,000
- 1 ≤ `pat`의 길이 ≤ 300
- `myString`과 `pat`은 모두 알파벳으로 이루어진 문자열입니다.

## 입출력 예
| myString | pat | return |
| :-- | :-- | :-- |
| "AbCdEfG" | "aBc" | 1 |
| "aaAA" | "aaaaa" | 0 |

## 입출력 예 설명
입출력 예 #1
- "AbCdEfG"의 0~2번 인덱스의 문자열은 "AbC"이며, 이는 `pat`인 "aBc"와 같습니다. 따라서 1을 return 합니다.

입출력 예 #2
- `myString`의 길이가 `pat`보다 더 짧기 때문에 `myString`의 부분 문자열 중 `pat`와 같은 문자열이 있을 수 없습니다. 따라서 0을 return 합니다.

## 배운점
- 문자열 검색 & 비교
    - 부분 검색 ( Boolean )
        - `contains(_:)`: 대소문자를 구분하여 문자열을 찾음
        - `localizedCaseInsensitiveContains(_:)`: 대소문자를 완전히 무시하며 문자열을 복사하지 않고 원본에서 바로 검색하여 성능이 뛰어나다.
        
    - 완전 일치 ( Boolean )
        - `compare(_:)`: 대소문자를 구분하여 완벽히 일치하는지 비교한다.
        - `caseInsensitiveCompare(_:)`: 대소문자를 무시하고 완벽히 일치하는지 비교한다.
        
    - 인덱스 검색
        - `range(of:)`: 단어가 존재하는 범위를 반환한다.
        - `range(of:options:)`: 옵션에 `.caseInsenitive`(대소문자 무시), `.backwards`(뒤에서 부터 찾기) 등의 옵션이 있다.
        
    - 문자열의 시작과 끝 추출
        - `hasPrefix(_:)`,`hasSuffix(_:)`: 특정한 단어로 시작/끝나는지 확인한다. ( Boolean )
        - `commonPrefix(with:)`,`commonSuffix(with:)`: 두개의 문자열을 비교하여 앞,뒤 에서부터 일치하는 공통된 문자열을 반환한다.

## 나의 풀이
```swift
func solution(_ myString:String, _ pat:String) -> Int {
    myString.localizedCaseInsensitiveContains(pat) ? 1 : 0
}
```
