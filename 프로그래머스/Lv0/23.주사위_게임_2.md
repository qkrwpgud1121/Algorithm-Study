# 주사위 게임 2

## 문제 설명
1부터 6까지 숫자가 적힌 주사위가 세 개 있습니다. 세 주사위를 굴렸을 때 나온 숫자를 각각 `a`, `b`, `c`라고 했을 때 얻는 점수는 다음과 같습니다.
- 세 숫자가 모두 다르다면 `a` + `b` + `c` 점을 얻습니다.
- 세 숫자 중 어느 두 숫자는 같고 나머지 다른 숫자는 다르다면 (`a` + `b` + `c`) × ($$a^2$$ + $$b^2$$ + $$c^2$$)점을 얻습니다.
- 세 숫자가 모두 같다면 (`a` + `b` + `c`) × ($$a^2$$ + $$b^2$$ + $$c^2$$ ) × ($$a^3$$ + $$b^3$$ + $$c^3$$ )점을 얻습니다.

세 정수 `a`, `b`, `c`가 매개변수로 주어질 때, 얻는 점수를 return 하는 solution 함수를 작성해 주세요.

## 제한사항
- a, b, c는 1이상 6이하의 정수입니다.

## 입출력 예
| a | b | c | result |
| :-- | :-- | :-- | :-- |
| 2 | 6 | 1 | 9 |
| 5 | 3 | 3 | 473 |
| 4 | 4 | 4 | 110592 |

## 입출력 예 설명
입출력 예 #1
- 예제 1번에서 세 주사위 숫자가 모두 다르므로 2 + 6 + 1 = 9점을 얻습니다. 따라서 9를 return 합니다.

입출력 예 #2
- 예제 2번에서 두 주사위 숫자만 같으므로 (5 + 3 + 3) × (52 + 32 + 32 ) = 11 × 43 = 473점을 얻습니다. 따라서 473을 return 합니다.

입출력 예 #3
- 예제 3번에서 세 주사위 숫자가 모두 같으므로 (4 + 4 + 4) × (42 + 42 + 42 ) × (43 + 43 + 43 ) = 12 × 48 × 192 = 110,592점을 얻습니다. 따라서 110592를 return 합니다.

## 나의 풀이
```swift
func solution(_ a:Int, _ b:Int, _ c:Int) -> Int {
    if a != b && b != c && a != c {
        return a + b + c
    } else if a == b && a == c && b == c {
        return (a + b + c) * ((a * a) + (b * b) + (c * c)) * ((a * a * a) + (b * b * b) + (c * c * c))
    } else {
        return (a + b + c) * ((a * a) + (b * b) + (c * c))
    }
}
```

## 이전 학습 내용 적용
- Switch Expression: switch문 자체가 값을 반환하는 식으로 사용될 수 있다는것을 기억하고 적용했다.

## 배운점
- pow(_ : _ :) -> Decimal
    - 수의 제곱을 계산해주는 메서드 이다.
    - Foundation 프레임워크에 정의 되어 있으며 Double 타입의 인자를 받는다.
    - 해당 문제에서 사용하지 않은이유는 정수 연산이 메인일때 타입 변환 과정이 복잡해지고 미세한 소수점 오차가 발생할 수 있으며 요번 처럼 상대적으로 짧은 곱셈 연산에 대해서는 효율성이 떨어질 수도 있다.
    
- Set
    - 중복제거 메서드 타입 추론으로 생성 불가 무조건 `[]`로 생성
    - 해당 문제에서 Set의 count 속성만으로 동일 하지 않은 주사위의 개수를 확인할 수 있다.
    
- if-else와 switch-case 의 차이
    - 조건이 많은 상황에서는 if-else문 보다 가독성이 좋다.
    - 분기 개수가 많아질 경우 switch의 내부적인 Jump Table 을 사용하여 빠르게 해당 케이스를 찾아갈 수. 있다.
    - if-else문은 시간 복잡도로 인해 성능에 단점이 있고 switch-case는 case의 개수만큼의 Jump Table을 차지하므로 메모리에 단점이 있다.

## 새로운 풀이
```swift
import Foundation

func solution(_ a:Int, _ b:Int, _ c:Int) -> Int {

    let count = Set([a,b,c]).count
        
    let sum1 = a + b + c
    let sum2 = (a * a) + (b * b) + (c * c)
    let sum3 = (a * a * a) + (b * b * b) + (c * c * c)
    
    return switch count {
    case 1: sum1 * sum2 * sum3
    case 2: sum1 * sum2
    case 3: sum1
    default: 0
    }
}
```
