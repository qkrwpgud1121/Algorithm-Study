# 글자 이어 붙여 문자열 만들기

## 문제 설명
문자열 `my_string`과 정수 배열 `index_list`가 매개변수로 주어집니다. `my_string`의 `index_list`의 원소들에 해당하는 인덱스의 글자들을 순서대로 이어 붙인 문자열을 return 하는 solution 함수를 작성해 주세요.

## 제한사항
- 1 ≤ my_string의 길이 ≤ 1,000
- my_string의 원소는 영소문자로 이루어져 있습니다.
- 1 ≤ index_list의 길이 ≤ 1,000
- 0 ≤ index_list의 원소 < my_string의 길이

## 입출력 예
| my_string | index_list | result |
| :-- | :-- | :-- |
| "cvsgiorszzzmrpaqpe" | [16, 6, 5, 3, 12, 14, 11, 11, 17, 12, 7] | "programmers" |
| "zpiaz" | [1, 2, 0, 0, 3] | "pizza" |

## 입출력 예 설명
입출력 예 #1
- 예제 1번의 `my_string`에서 인덱스 3, 5, 6, 11, 12, 14, 16, 17에 해당하는 글자는 각각 g, o, r, m, r, a, p, e이므로 `my_string`에서 `index_list`에 들어있는 원소에 해당하는 인덱스의 글자들은 각각 순서대로 p, r, o, g, r, a, m, m, e, r, s입니다. 따라서 "programmers"를 return 합니다.

입출력 예 #2
- 예제 2번의 `my_string`에서 인덱스 0, 1, 2, 3에 해당하는 글자는 각각 z, p, i, a이므로 `my_string`에서 `index_list`에 들어있는 원소에 해당하는 인덱스의 글자들은 각각 순서대로 p, i, z, z, a입니다. 따라서 "pizza"를 return 합니다.

## 나의 풀이

### 첫 번째 
- 문제에 대한 직관적으로 코드를 작성했다.
- `reduce` 루프 안에서 매번 my_string을 배열로 변환하고 있다.
```swift
func solution(_ my_string:String, _ index_list:[Int]) -> String {
    return index_list.map { $0 }.reduce(into: "") { (result, index) in
        result += my_string.map { String($0) }[index]
    }
}
```

### 두 번째
- 문자열을 배열로 변환하는 작업을 `reduce` 루프 밖으로 빼 한번만 실행하도록 했다.
- 이전 문제에서 공부했었던 쓸모없는 `.map { $0 }` 코드를 제거하였다.
```swift
func solution(_ my_string:String, _ index_list:[Int]) -> String {

    let charArray = Array(my_string)
    
    return index_list.reduce(into: "") { result, index in 
        result += String(charArray[index])
    }
}
```

### 세 번째
- index_list의 숫자를 my_string의 글자로 map을 사용하여 변환 후에 String으로 결합하도록 수정했다.
```swift
func solution(_ my_string:String, _ index_list:[Int]) -> String {

    let charArray = Array(my_string)
    
    return String(index_list.map { charArray[$0] })
}
```

## 배운점
- 문자열에서는 인덱스로 접근하기 어려워 문자열을 배열로 변환 후에 접근하였다.
- 반복문, 고차 함수 내부에서는 변하지 않는 무거운 연산 및 타입 변환에 대해서는 반드시 루프 밖으로 빼 한번만 실행 되도록 해야 한다.
- *데이터를 어떻게 하나씩 꺼내어 붙일지 말고 데이터의 형태를 어떻게 변환할 것인지에 집중해야한다.*
