# 문자열 섞기

## 문제 설명
길이가 같은 두 문자열 `str1`과 `str2`가 주어집니다.

두 문자열의 각 문자가 앞에서부터 서로 번갈아가면서 한 번씩 등장하는 문자열을 만들어 return 하는 solution 함수를 완성해 주세요.

## 제한사항
- 길이가 같은 두 문자열 str1과 str2가 주어집니다.
    - 두 문자열의 각 문자가 앞에서부터 서로 번갈아가면서 한 번씩 등장하는 문자열을 만들어 return 하는 solution 함수를 완성해 주세요.

## 입출력 예
str1 | str2 | result
"aaaaa" | "bbbbb" | "ababababab"

## 나의 풀이
```swift
func solution(_ str1:String, _ str2:String) -> String {
    
    let str11 = str1.map { $0 }
    let str22 = str2.map { $0 }
    var result: String = ""
        
    for i in 0..<str1.count {
        result.append(str11[i])
        result.append(str22[i])
    }
        
    return result
}
```

## 배운점
- **zip**: 두개의 시퀸스를 하나로 묶어 튜플의 시퀸스로 만들어주는 Swift 내장 함수이다.
- 두 문자열의 길이가 같을때 for 루프 대신 zip을 사용하면 인덱스 참조 오류 위험을 줄일 수 있다.
- String($0) 보다 \($0) 을 사용하는 것이 매번 새로운 문자열을 생성하지 않는다는 것을 알 수 있었다.

## 새로운 풀이
```swift
func solution(_ str1:String, _ str2:String) -> String {
    return zip(str1, str2).map { "\($0)\($1)" }.joined()
}
```
