#

## 문제 설명
문자들이 담겨있는 배열 `arr`가 주어집니다. `arr`의 원소들을 순서대로 이어 붙인 문자열을 return 하는 solution함수를 작성해 주세요.

## 제한사항
- 1 ≤ `arr`의 길이 ≤ 200
    - `arr`의 원소는 전부 알파벳 소문자로 이루어진 길이가 1인 문자열입니다.
    
## 입출력 예
| arr | result |
| :-- | :-- |
| ["a","b","c"] | "abc" |

## 나의 풀이
```swift
func solution(_ arr:[String]) -> String {
    arr.joined()
}
```

## 이전 학습 내용 적용
- 직전 문제에서 사용했던 `joined()`를 사용하여 문제를 해결했다.
- `joined(separator:)` 의 separator 를 지정하지 않고 사용하면 배열의 각 요소 사이에 아무런 공백없이 그대로 이어 붙여진다.

## 배운점
- 직전 문제에서 배운것이지만 다시 복기를 해보면 `for` 루프를 돌며 `+` 연산자로 이어 붙이는 것은 매번 새롭게 메모리를 할당할 수 있지만 `joined()`를 사용하면 합쳐질 문자열의 길이를 미리 계산하여 효율적으로 동작한다는 점을 다시 생각해보았다.
- 만약 배열의 각 요소 사이에 공백이나 쉼표가 필요하면 `joined(separator: " ")` 처럼 파라미터를 추가하여 붙일수있다.
