# n 번째 원소까지

## 문제 설명
정수 리스트 `num_list`와 정수 `n`이 주어질 때, `num_list`의 첫 번째 원소부터 `n` 번째 원소까지의 모든 원소를 담은 리스트를 return하도록 solution 함수를 완성해주세요.

## 제한사항
- 2 ≤ `num_list`의 길이 ≤ 30
- 1 ≤ `num_list`의 원소 ≤ 9
- 1 ≤ `n` ≤ `num_list`의 길이

## 입출력 예
| num_list | n | result |
| :-- | :-- | :-- |
| [2, 1, 6] | 1 | [2] |
| [5, 2, 1, 7, 5] | 3 | [5, 2, 1] |

## 입출력 예 설명
입출력 예 #1
- [2, 1, 6]의 첫 번째 원소부터 첫 번째 원소까지의 모든 원소는 [2]입니다.

입출력 예 #2
- [5, 2, 1, 7, 5]의 첫 번째 원소부터 세 번째 원소까지의 모든 원소는 [5, 2, 1]입니다.

## 배운점
- 범위연산자, prefix의 차이
    - 범위 연산자는 배열의 길이보다 길게 자르려고 하면 Index of Range 오류를 발생하며 강제 종료 시킨다.
    - prefix는 배열의 길이보다 길게 자르려고 하면 에러를 발생시키지 않고 가지고 있는 것만 반환하여 범위 연산자 보다 안전성에서 뛰어나다.

## 나의 풀이

### 첫번째 풀이
```swift
func solution(_ num_list:[Int], _ n:Int) -> [Int] {
    Array(num_list[0...(n-1)])
}
```

### 두번째 풀이
```swift
func solution(_ num_list:[Int], _ n:Int) -> [Int] {
    Array(num_list.prefix(n))
}
```
