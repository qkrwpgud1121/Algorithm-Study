# 조건에 맞게 수열 변환하기 2

## 문제 설명
정수 배열 `arr`가 주어집니다. `arr`의 각 원소에 대해 값이 50보다 크거나 같은 짝수라면 2로 나누고, 50보다 작은 홀수라면 2를 곱하고 다시 1을 더합니다.

이러한 작업을 `x`번 반복한 결과인 배열을 `arr(x)`라고 표현했을 때, `arr(x) = arr(x + 1)`인 `x`가 항상 존재합니다. 이러한 `x` 중 가장 작은 값을 return 하는 solution 함수를 완성해 주세요.

단, 두 배열에 대한 "="는 두 배열의 크기가 서로 같으며, 같은 인덱스의 원소가 각각 서로 같음을 의미합니다.

## 제한사항
- 1 ≤ `arr`의 길이 ≤ 1,000,000
- 1 ≤ `arr`의 원소의 값 ≤ 100

## 입출력 예
| arr | result |
| :-- | :-- |
| [1, 2, 3, 100, 99, 98] | 5 |

## 입출력 예 설명
입출력 예 #1
- 위 작업을 반복하면 다음과 같이 `arr`가 변합니다.

| 반복 횟수 | arr |
| :-- | :-- |
| 0 | [1, 2, 3, 100, 99, 98] |
| 1 | [3, 2, 7, 50, 99, 49] | 
| 2 | [7, 2, 15, 25, 99, 99] |
| 3 | [15, 2, 31, 51, 99, 99] |
| 4 | [31, 2, 63, 51, 99, 99] |
| 5 | [63, 2, 63, 51, 99, 99] |
| 6 | [63, 2, 63, 51, 99, 99] | 

- 이후로 `arr`가 변하지 않으며, `arr(5) = arr(6)`이므로 5를 return 합니다.

## 배운점
- 반복 횟수가 정해져있지 않고 특정 조건을 만족할때 까지 돌아야 하는경우 `while true`와 `break`를 사용하면 된다는것을 상기시켰다.
- 직점 문제에서 공부했던 `map`을 사용해서 원본 배열과 변환된 배열을 비교해 `break`를 사용하여 무한루프를 종료할 수 있었다.

- 데이터가 많을때 무작정 배열 전체를 변환하는 `map`,`for-in`을 사용하면 많은 메모리 사용량과 비교 연산을 발생시킨다는 것을 알게되었다.
- 해서 최종 반환값이 Input과 달리 하나의 숫자, 배열의 각각 요소가 서로 영향을 주지 않는 독립적인 상태, 배열 각각의 요소의 변환 횟수가 제각각일때 **배열 전체를 매번 비교하지 않고 배열 각각의 요소를 꺼내어 변환하고 그중 최대 변환값을 추출**하는 방법으로 시간과 메모리를 줄일수 있다는것을 알게되었다.


## 나의 풀이

### 첫번째 풀이
```swift
func solution(_ arr:[Int]) -> Int {
    var currentArr = arr
    var count = 0
    
    while true {
        
        let beforeArr = currentArr
        
        for (index, value) in currentArr.enumerated() {
            if value >= 50 && value.isMultiple(of: 2) {
                currentArr[index] = value / 2
            } else if value < 50 && !value.isMultiple(of: 2) {
                currentArr[index] = value * 2 + 1
            }
        }
        
        if beforeArr == currentArr {
            break
        }
        
        count += 1
    }
    
    return count
}
```

### 두번째 풀이
```swift
func solution(_ arr:[Int]) -> Int {
    var currentArr = arr
    var count = 0
    
    while true {
        let newArr = currentArr.map { value in
            if value >= 50 && value.isMultiple(of: 2) {
                return value / 2
            } else if value < 50 && !value.isMultiple(of: 2) {
                return value * 2 + 1
            } else {
                return value
            }
        }
        
        if currentArr == newArr { break }
        
        currentArr = newArr
        count += 1
    }
    
    return count
}
```

### 세번째 풀이
```swift
func solution(_ arr:[Int]) -> Int {
    var maxCount = 0
    
    for number in arr {
        var currentNum = number
        var count = 0
        
        while true {
            if currentNum >= 50 && currentNum.isMultiple(of: 2) {
                currentNum = currentNum / 2
                count += 1
            } else if currentNum < 50 && !currentNum.isMultiple(of: 2) {
                currentNum = currentNum * 2 + 1
                count += 1
            } else {
                break
            }
        }
        
        if count > maxCount {
            maxCount = count
        }
    }
    
    return maxCount
}
```
