# 조건에 맞게 수열 변환하기 1

## 문제 설명
정수 배열 `arr`가 주어집니다. `arr`의 각 원소에 대해 값이 50보다 크거나 같은 짝수라면 2로 나누고, 50보다 작은 홀수라면 2를 곱합니다. 그 결과인 정수 배열을 return 하는 solution 함수를 완성해 주세요.

## 제한사항
- 1 ≤ `arr`의 길이 ≤ 1,000,000
- 1 ≤ `arr`의 원소의 값 ≤ 100

## 입출력 예
| arr | result |
| :-- | :-- |
| [1, 2, 3, 100, 99, 98] | [2, 2, 6, 50, 99, 49] |

## 입출력 예 설명
입출력 예 #1
- 1, 3은 50 미만의 홀수 이므로 2를 곱하고, 100, 98은 50 이상의 짝수이므로 2로 나눕니다.
- 나머지 값들은 변경 조건에 해당하지 않으므로 바꾸지 않습니다.
- 따라서 [2, 2, 6, 50, 99, 49]를 return 합니다.

## 나의 풀이
```swift
func solution(_ arr:[Int]) -> [Int] {
    return arr.map { value in
        if value >= 50 && value.isMultiple(of: 2) {
            return value / 2
        } else if value < 50 && !value.isMultiple(of: 2) {
            return value * 2
        } else {
            return value
        }
    }
}
```

## 풀이 이유
1. Input과 Output의 1:1 Mapping 구조라고 판단, `map` 함수가 가장 잘 맞는다고 생각하였다.
2. 외부 변수의 상태를 변경하거나 복잡한 로직을 처리하는게 아닌 단순 조건에 맞춰 값을 가공해야하여 외부환경을 관여하지 않는 `map` 을 선택하였다.
3. 코드를 짧게 보이기 위해 억지로 삼항연산자를 사용하지 않았으며 `value in`과 `if-else`를 사용하여 조건들이 명확히 보일수있도록 하였다.
