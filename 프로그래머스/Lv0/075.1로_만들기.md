# 1로 만들기

## 문제 설명
정수가 있을 때, 짝수라면 반으로 나누고, 홀수라면 1을 뺀 뒤 반으로 나누면, 마지막엔 1이 됩니다. 예를 들어 10이 있다면 다음과 같은 과정으로 1이 됩니다.

- 10 / 2 = 5
- (5 - 1) / 2 = 2
- 2 / 2 = 1
위와 같이 3번의 나누기 연산으로 1이 되었습니다.

정수들이 담긴 리스트 `num_list`가 주어질 때, `num_list`의 모든 원소를 1로 만들기 위해서 필요한 나누기 연산의 횟수를 return하도록 solution 함수를 완성해주세요.

## 제한사항
- 3 ≤ `num_list`의 길이 ≤ 15
- 1 ≤ `num_list`의 원소 ≤ 30

## 입출력 예
| num_list | result |
| :-- | :-- |
| [12, 4, 15, 1, 14] | 11 |

## 입출력 예 설명
입출력 예 #1
- 12는 3번, 4는 2번, 15는 3번, 1은 0번, 14는 3번의 연산이 필요하기 때문에 총 11번의 연산이 필요합니다.

## 배운점
- 이전 문제에서 공부했던 배열 각각의 요소를 따로 계산하는 조건에 해당하여 `map`을 사용하지 않고 문제를 해결하였다.
- 기존 아무생각없이 사용하던 `+=`이 복합 할당 연산자 라는것을 `/=`를 공부하면서 알게되었다.
- Swift 정수 나눗셈은 정수이면서 양수일경우 소수점을 버린다는 특징으로 인해 `(홀수 - 1) / 2`와 `홀수 / 2`의 값이 같다는것을 알게 되어 불필요한 `if-else`를 제거 할수 있었다.

## 나의 풀이

### 첫번째 풀이
```swift
func solution(_ num_list:[Int]) -> Int {
    
    var count = 0
    
    for num in num_list {
        
        var currentNum = num
        
        while true {
            
            if currentNum == 1 { break }
            
            if currentNum.isMultiple(of: 2) {
                currentNum = currentNum / 2
                count += 1
            } else {
                currentNum = (currentNum - 1) / 2
                count += 1
            }
        }
    }
    
    return count
}
```

### 두번째 풀이
```swift
func solution(_ num_list:[Int]) -> Int {
    
    var count = 0
    
    for num in num_list {
        
        var currentNum = num
        
        while currentNum > 1 {
            currentNum /= 2
            count += 1
        }
    }
    
    return count
}
```
