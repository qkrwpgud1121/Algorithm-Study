# 문제 설명
정수로 이루어진 문자열 `n_str`이 주어질 때, `n_str`의 가장 왼쪽에 처음으로 등장하는 0들을 뗀 문자열을 return하도록 solution 함수를 완성해주세요.

## 제한사항
- 2 ≤ `n_str` ≤ 10
- `n_str`이 "0"으로만 이루어진 경우는 없습니다.

## 입출력 예

| n_str | result |
| :-- | :-- |
| "0010" | "10" |
| "854020" | "854020" |

## 입출력 예 설명

입출력 예 #1
- "0010"의 가장 왼쪽에 연속으로 등장하는 "0"을 모두 제거하면 "10"이 됩니다.

입출력 예 #2
- "854020"는 가장 왼쪽에 0이 없으므로 "854020"을 return합니다.

## 배운점
- `Int()`로 형변을 하여 정수로 변환하면 앞의 0들은 지워진다는 것을 알게 되었다.
- 하지만 파라미터로 받은 문자열의 길이가 정수형 메모리 한계인 64비트를 초과할 경우 끝까지 읽지 못하거나 앱의 크래시가 발생한다.
- 해서 배열로 변환하여 인덱스를 찾아 대형 문자열이 들어와도 괜찮은 방식으로 문제를 해결하였다.

- 문자열 자체에서 처음으로 특정 조건에 부합하는 인덱스를 찾을 수 있는 `firstIndex(where:)`를 알게 되었다.
- 이를 통해 문자열을 `[Int]`나 `[String]` 배열로 여러번 생성하지 않고 원본 문자열의 고유 인덱스 범위만 잘라내는 `suffix(from:)`을 사용하여 메모리 낭비를 하지 않는 방법을 알게 되었다.

## 나의 풀이

### 첫번째 풀이
```swift
func solution(_ n_str:String) -> String {
    
    let intArr = Array(n_str.compactMap { $0.wholeNumberValue })
    var startIndex = 0
    
    for (offset, value) in intArr.enumerated() {
        if value != 0 {
            startIndex = offset
            break
        }
    }
    
    return intArr[startIndex...].map { String($0)}.joined()
}
```

### 두번째 풀이
```swift
func solution(_ n_str:String) -> String {
    
    if let startIndex = n_str.firstIndex(where: { $0 != "0" }) {
        return String(n_str.suffix(from: startIndex))
    }
    
    return n_str
}
```
