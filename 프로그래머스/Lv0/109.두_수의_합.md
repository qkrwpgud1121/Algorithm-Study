# 두 수의 합

## 문제 설명
0 이상의 두 정수가 문자열 `a`, `b`로 주어질 때, `a` + `b`의 값을 문자열로 return 하는 solution 함수를 작성해 주세요.

## 제한사항
- 1 ≤ `a`의 길이 ≤ 100,000
- 1 ≤ `b`의 길이 ≤ 100,000
- `a`와 `b`는 숫자로만 이루어져 있습니다.
- `a`와 `b`는 정수 0이 아니라면 0으로 시작하지 않습니다.

## 입출력 예
| a | b | result |
| :-- | :-- | :-- |
| "582" | "734" | "1316" |
| "18446744073709551615" | "287346502836570928366" | "305793246910280479981" |
| "0" | "0" | "0" |

## 입출력 예 설명
입출력 예 #1
- 예제 1번의 `a`, `b`는 각각 582, 734이고 582 + 734 = 1316입니다. 따라서 "1316"을 return 합니다.

입출력 예 #2
- 예제 2번의 `a`, `b`는 각각 18446744073709551615, 287346502836570928366이고 18446744073709551615 + 287346502836570928366 = 305793246910280479981입니다. 따라서 "305793246910280479981"을 return 합니다.

입출력 예 #3
- 예제 3번의 `a`, `b`는 각각 0, 0이고 0 + 0 = 0입니다. 따라서 "0"을 return 합니다.

## 배운점
- 문제의 제한 사항의 10만자리의 문자열 길이를 보고 일반적으로 사용하는 `Int`로는 연산이 불가능 한 오버플로우 문제인 것을 느꼈다.
- `popLast()`, `.wholeNumberValue`를 사용하여 각각의 문자열의 끝 수를 가지고와 숫자로 변환하여 연산을 하고 `10`이 넘어가면 `1`을 다음자릿 수로 이월시키는 세로셈을 알게 되었다.
- Swift 에서는 `BigInt` 라이브러리를 가져와 연산할수 있다는 것을 알게 되었다.

## 나의 풀이

### 첫번째 풀이
```swift
func solution(_ a:String, _ b:String) -> String {
    
    var arrA = Array(a), arrB = Array(b)
    var carry = 0
    var result = ""

    while !arrA.isEmpty || !arrB.isEmpty || carry > 0 {
        
        let numA = arrA.popLast()?.wholeNumberValue ?? 0
        let numb = arrB.popLast()?.wholeNumberValue ?? 0
        
        let sum = numA + numb + carry
        
        carry = sum / 10
        
        result += String(sum % 10)
    }
    
    return String(result.reversed())
}
```

### 두번째 풀이
```swift
func solution(_ a:String, _ b:String) -> String {

    let bigA = BigInt(a) ?? 0
    let bigB = BigInt(b) ?? 0
    
    return String(bigA+bigB)
}
```
