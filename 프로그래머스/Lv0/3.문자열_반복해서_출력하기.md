# 문자열 반복해서 출력하기

## 문제 설명
문자열 `str`과 정수 `n`이 주어집니다.
`str`이 `n`번 반복된 문자열을 만들어 출력하는 코드를 작성해 보세요.

## 제한사항
- 1 ≤ `str`의 길이 ≤ 10
- 1 ≤ `n` ≤ 5

## 입출력 예

**입력 #1**
```
string 5
```
**출력 #1**
```
stringstringstringstringstring
```
---

## 나의 풀이
```swift
let inp = readLine()!.split(separator: " ").map { $0 }
let (s1, a) = (inp[0], Int(inp[1])!)

var str = ""

for i in 1...a {
    str += s1
}

print(str)
```

## 배운점
- 직전 문제에서 배웠던 components(separatedBy:) 대신 split(separator:)을 사용하였다.
- for 문을 대체하여 Swift 내장 함수인 String(repeating:count:) 을 사용할 수 있다는것을 배웠다.
- .map 에 대해 공부하다가 .map { $0 } 은 아무런 변화가 없기때문에 해당 문제에서는 쓸모가 없다는것을 알았다.

## 새로운 풀이
```swift
let inp = readLine()!.split(separator: " ")
print(String(repeating: String(inp[0]), count: Int(inp[1])!))
```
