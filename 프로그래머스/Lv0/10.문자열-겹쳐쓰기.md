# 문자열 겹쳐쓰기

## 문제 설명
문자열 `my_string`, `overwrite_string`과 정수 `s`가 주어집니다. 문자열 `my_string`의 인덱스 `s`부터 `overwrite_string`의 길이만큼을 문자열 `overwrite_string`으로 바꾼 문자열을 return 하는 solution 함수를 작성해 주세요.

## 제한사항
- `my_string`와 `overwrite_string`은 숫자와 알파벳으로 이루어져 있습니다.
- 1 ≤ `overwrite_string`의 길이 ≤ `my_string`의 길이 ≤ 1,000
- 0 ≤ `s` ≤ `my_string`의 길이 - `overwrite_string`의 길이

## 입출력 예
| my_string | overwrite_string | s | result |
| :--- | :--- | :--- | :--- |
|"He11oWor1d" | "lloWorl" | 2 | "HelloWorld" |
|"Program29b8UYP" | "merS123" | 7 | "ProgrammerS123" |

## 입출력 예 설명
**입출력 예 #1**

    - 예제 1번의 my_string에서 인덱스 2부터 overwrite_string의 길이만큼에 해당하는 부분은 "11oWor1"이고 이를 "lloWorl"로 바꾼 "HelloWorld"를 return 합니다.
**입출력 예 #2**

    - 예제 2번의 my_string에서 인덱스 7부터 overwrite_string의 길이만큼에 해당하는 부분은 "29b8UYP"이고 이를 "merS123"로 바꾼 "ProgrammerS123"를 return 합니다.

## 나의 풀이
```swift
func solution(_ my_string:String, _ overwrite_string:String, _ s:Int) -> String {

    let count = overwrite_string.count + s

    let startIndex = my_string.index(my_string.startIndex, offsetBy: 0)
    let endIndex = my_string.index(my_string.startIndex, offsetBy: s)
    let subString = my_string[startIndex..<endIndex]
    let startIndex2 = my_string.index(my_string.startIndex, offsetBy: count)
    let endIndex2 = my_string.index(my_string.endIndex, offsetBy: 0)
    let subString2 = my_string[startIndex2..<endIndex2]

    return "\(subString)\(overwrite_string)\(subString2)"
}
```

## 배운점 1
- prefix
    - (_ maxLength: Int) : Int값으로 지정한 maxLength 까지 반환
    - (upTo: Int) : Int로 지정한 Index값을 제외 하여 반환
    - (through: Int) : Int로 지정한 Index값을 포함 하여 반환
    - (while predicate: (Element) -> Bool) : Element값에서 지정한 식의 false가 나올때까지 반환
- suffix
    - (_ maxLength: Int) : Int값으로 지정한 maxLength 까지 뒤에서 반환
    - (from start: Int) : Int로 지정한 Index값에서 끝까지 반환
- 범위를 지정할때 `[..<]`와 같은 `subscript notation`을 사용하는것이 더 선호되는 표기법이다.

## 적용 풀이
```swift
func solution(_ my_string:String, _ overwrite_string:String, _ s:Int) -> String {
    
    let a = Array(my_string)[..<s]
    let b = my_string.count - overwrite_string.count - s
    let c = Array(overwrite_string)[b...]
        
    return "\(String(a) + String(c))"
}
```

## 배운점 2
- 문자열을 잘라서 붙이는 것보다 문자열의 특정 범위를 교체하는 `replaceSubrange(_: with:)` 메서드가 있다는것을 알았으며 더욱 효율적이고 가독성이 좋다는것을 알았다.
- replaceSubrange를 사용하면 위에서 했던 풀이의 문자열의 자르고 붙이는 과정에서 새로운 문자열 메모리 공간을 생성하는 과정이 없기때문에 효율적이다.

## 적용 풀이
```swift
func solution(_ my_string:String, _ overwrite_string:String, _ s:Int) -> String {
    
    var myString = my_string
    
    let startIndex = myString.index(myString.startIndex, offsetBy: s)
    let endIndex = myString.index(startIndex, offsetBy: overwrite_string.count)
        
    myString.replaceSubrange(startIndex..<endIndex, with: overwrite_string)
        
    return myString
}
```
