# 첫 번째로 나오는 음수

## 문제 설명
정수 리스트 `num_list`가 주어질 때, 첫 번째로 나오는 음수의 인덱스를 return하도록 solution 함수를 완성해주세요. 음수가 없다면 -1을 return합니다.

## 제한사항
- 5 ≤ `num_list`의 길이 ≤ 100
- -10 ≤ `num_list`의 원소 ≤ 100

## 입출력 예
| num_list | result |
| :-- | :-- |
| [12, 4, 15, 46, 38, -2, 15] | 5 |
| [13, 22, 53, 24, 15, 6] | -1 |

## 입출력 예 설명
입출력 예 #1
- 5번 인덱스에서 음수가 처음 등장하므로 5를 return합니다.

입출력 예 #2
- 음수가 없으므로 -1을 return합니다.

## 배운점
- 전에 학습했던 `firstIndex(of:)`를 기억해 자동완성 목록을 살펴보다가 `firstIndex(where:)`라는 것을 발견하여 뭔가 조건식을 넣을수 있을거같아 선택한 후에 직전까지 주구장창 사용했던 클로저를 사용할 수 있을거같아 사용해 문제를 해결하였다.
- `firstIndex(of:)`는 정확히 일치하는 하나의 값을 찾을때 사용하지만 `firstIndex(where:)`는 클로저를 통해 조건식에 만족하는 첫번째 요소의 인덱스를 찾아낸다 라는것을 알게되었다.

## 나의 풀이
```swift
func solution(_ num_list:[Int]) -> Int {
    num_list.firstIndex(where: { $0 < 0 } ) ?? -1
}
```
