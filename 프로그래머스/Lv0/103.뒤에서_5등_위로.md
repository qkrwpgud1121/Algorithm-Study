# 뒤에서 5등 위로

## 문제 설명
정수로 이루어진 리스트 `num_list`가 주어집니다. `num_list`에서 가장 작은 5개의 수를 제외한 수들을 오름차순으로 담은 리스트를 return하도록 solution 함수를 완성해주세요.

## 제한사항
- 6 ≤ `num_list`의 길이 ≤ 30
- 1 ≤ `num_list`의 원소 ≤ 100

## 입출력 예

| num_list | result |
| :-- | :-- |
| [12, 4, 15, 46, 38, 1, 14, 56, 32, 10] | [15, 32, 38, 46, 56] |

## 입출력 예 설명

입출력 예 #1
- [12, 4, 15, 46, 38, 1, 14, 56, 32, 10]를 정렬하면 [1, 4, 10, 12, 14, 15, 32, 38, 46, 56]이 되고, 앞에서 부터 6번째 이후의 수들을 고르면 [15, 32, 38, 46, 56]가 됩니다.

## 배운점
- `.suffix(from:)`를 사용했을때 `Fatal error: Index out of range`가 발생하여 앱이 종료가 될 가능성이 있다.
- 때문에 정확한 인덱스를 구한 상황에서는 `.suffix`를 사용하는것이 좋지만 그렇지 않은 경우에는 `dropFirst`를 사용하는 것이 나은 선택이다.
- 실무에서는 앱 크래시 보다는 기능적 장애가 더 나은 선택지 인것 같다.
- 또한 가독성 면에서도 `.suffix`보다는 `dropFirst`가 더 직관적이다.

## 나의 풀이

### 첫번째 풀이
```swift
func solution(_ num_list:[Int]) -> [Int] {
    Array(num_list.sorted().suffix(from: 5))
}
```

### 두번째 풀이
```swift
func solution(_ num_list:[Int]) -> [Int] {
    Array(num_list.sorted().dropFirst(5))
}
```
