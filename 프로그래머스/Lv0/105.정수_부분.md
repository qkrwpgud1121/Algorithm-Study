# 정수 부분

## 문제 설명
실수 `flo`가 매개 변수로 주어질 때, `flo`의 정수 부분을 return하도록 solution 함수를 완성해주세요.

## 제한사항
- 0 ≤ `flo` ≤ 100

## 입출력 예

| flo | result |
| :-- | :-- |
| 1.42 | 1 |
| 69.32 | 69 |

## 입출력 예 설명

입출력 예 #1
- 1.42의 정수 부분은 1입니다.

입출력 예 #2
- 69.32의 정수 부분은 69입니다.

## 배운점
- 소수점 아래 부분을 버리는 방법이 `Int()`, `trunc`, `floor`가 있다는 것을 알게 되었다.
- `Int()`는 `Double`형태의 수에서 소수점 아래를 버리고 `Int`로 형태를 변경할때 사용한다.
- `trunc`는 `Double`형태의 수에서 소수점 아래를 버리고 `Double`형태를 유지할때 사용한다.
- `floor`는 `Double`형태를 유지하지만 수학적으로 아래의 수로 내림을 할때 사용한다.

## 나의 풀이
```swift
func solution(_ flo:Double) -> Int {
    Int(flo)
}
```
