# 대소문자 바꿔서 출력하기

## 문제 설명
영어 알파벳으로 이루어진 문자열 `str`이 주어집니다. 
각 알파벳을 대문자는 소문자로 소문자는 대문자로 변환해서 출력하는 코드를 작성해 보세요.

## 제한사항
* 1 ≤ `str`의 길이 ≤ 20
    - str은 알파벳으로 이루어진 문자열입니다.

## 입출력 예

**입력 #1**
```
aBcDeFg
```
**출력 #1**
```
AbCdEfG
```
---

## 나의 풀이
```swift
import Foundation

let s1 = readLine()!
var result = ""

for char in s1 {
    if char.isLowercase {
        result += char.uppercased()
    } else {
        result += char.lowercased()
    }
}

print(result)
```

## 배운점
- for문의 readLine으로 입력받은 문자열을 반복할수 있어서 바로 사용할 수 있다.
- 직전에서 배웠던 .map 을 사용해서 새로운 변수에 담지 않고 변경할수 있으며 joined로 다시 문자열로 변환한다.
- if 문을 삼항연산자를 사용할수도 있다.

## 새로운 풀이
```swift
print(readLine()!.map { $0.isLowercase ? $0.uppercased() : $0.lowercased()}.joined())
```
